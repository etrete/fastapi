# Delivery Service API

Микросервис для расчёта стоимости доставки посылок.  
Быстрое API на FastAPI с поддержкой MySQL и асинхронных вычислений.

---

## Возможности

- **Регистрация посылки** — создание, сохранение типа, веса, стоимости и других параметров.
- **Получение и фильтрация посылок** — поддержка пагинации, фильтров по типу, стоимости, флагу "рассчитано" и другим атрибутам.
- **Расчёт стоимости доставки** — с учётом типа, веса, содержимого, а также текущих курсов валют (USD/RUB).
- **Пользовательские сессии** — все действия и получение посылок по уникальной сессии (без авторизации, через cookie).
- **Работа с типами посылок и администрацией** — CRUD для типов, административные маршруты.
- **Асинхронная работа с базой и сетевыми сервисами** — высокая производительность.

---

## Быстрый старт

docker-compose up --build

http://localhost:8080/docs для Swagger UI

---

## Структура API (основное):

- `POST /api/v1/packages/` — новая посылка
- `GET /api/v1/packages/` — список посылок c пагинацией/фильтрами
- `GET /api/v1/packages/{id}` — информация о посылке
- `GET /api/v1/package-types/` — типы посылок
- `GET /health` — проверка статуса сервиса
